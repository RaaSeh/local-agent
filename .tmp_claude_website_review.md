# Recommendations

## Key Improvements

### pwasher agent
- **Add website update output block** – Weekly content/page updates for rocketwashbc.com tailored to industrial/commercial pivot
- **Strengthen evidence requirement** – Demand Google Maps links, business registries, or LinkedIn profiles; reject vague "likely" claims
- **Reduce output size** – Drop from 8 to 5 leads; emphasize quality over quantity for a 2-person team
- **Add CTA/proof requirements** – Each website recommendation must specify where proof photos or testimonials go
- **Remove redundant devil's-advocate** – Let supervisor own all critique (cleaner separation of concerns)

### supervisor agent
- **Structured rejection criteria** – Explicit checklist: no address = reject, no evidence URL = reject, residential indicators = reject
- **Website review block** – Validate pwasher's Wix update recommendations against brand voice, technical feasibility, conversion best-practices
- **Capacity gating** – Flag when call list or website tasks exceed 2-person bandwidth in a given week
- **Evidence URL verification** – Require supervisor to spot-check at least 2 links per list
- **Actionable feedback loop** – Must provide 3 specific edits to pwasher prompt for next cycle
